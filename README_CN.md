# 开始

另一个完全基于Flutter框架的待办事项列表应用程序。

您可以从源代码构建应用程序，也可以快速下载[已发布的apk](https://github.com/Rogerskelamen/flodo/releases)（仅提供Android版本）

如果您想以自己的方式修改应用程序，非常欢迎，祝您愉快！
